# mall-customer-data-cleaning
Objective:
Clean and preprocess the Mall Customer Segmentation dataset to prepare it for analysis and modeling.

Tools Used: Python, Pandas, Jupyter Notebook

Data Cleaning Steps Performed:
Verified dataset structure using .info()
Checked for missing values using .isnull() (No missing values found)
Checked for duplicate records using .duplicated() (No duplicates found)
Standardized column names (converted to lowercase, removed spaces/special characters)
Validated data types

Outcome: The dataset is clean, consistent, and ready for exploratory data analysis or machine learning applications.
